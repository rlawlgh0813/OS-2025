# OS Practice 2025

운영체제(OS) 과목 과제 대비를 위한 연습 저장소입니다.  
Linux 환경에서 C, 시스템콜, 프로세스, 스레드, 메모리 관리 등 핵심 개념을 실습합니다.

---

## 개발 환경

- OS: Ubuntu 22.04 LTS (VMware, M2 Mac)
- Compiler: gcc 11.4.0
- Editor: Vim
- Git: SSH 연동

---

## 디렉토리 구조

- `lab01/` : 프로세스 생성 및 관리 실습
- `lab02/` : 스레드 및 동기화 실습
- `lab03/` : 메모리 관리 및 가상 메모리 실습
